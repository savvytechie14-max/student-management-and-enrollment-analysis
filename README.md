# student-management-and-enrollment-analysis

Tech Studio Student Data Analysis (Power BI Project)
## Project Overview

This project presents an interactive Power BI dashboard developed to analyze student enrollment data at Tech Studio. The goal of the analysis is to provide clear insights into course participation, student distribution, and overall enrollment patterns using visual analytics.

The dashboard transforms raw data into meaningful visualizations that help stakeholders quickly understand trends and make informed decisions regarding course offerings and student engagement.

## Objectives

* Analyze student enrollment across different courses.

* Identify the most popular courses based on student participation.

* Present data in an interactive and easy-to-interpret format.

* Support data-driven decision-making for academic planning.

## Tools Used

Power BI – Data visualization and dashboard development

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Creating calculated measures and metrics

## Key Features of the Dashboard

Interactive charts showing student distribution across courses

Visual summaries of total enrollments

Clean and intuitive dashboard layout for quick insights

Dynamic filtering to explore different segments of the dataset

## Insights

The dashboard highlights enrollment patterns and helps identify which courses attract the highest number of students. These insights can support better planning of training resources and marketing strategies for future cohorts.

## Conclusion

This project demonstrates the use of Power BI for transforming raw educational data into actionable insights through effective data modeling and visualization.
